📚 Panduan & Deploy
Cara menggunakan dan mengembangkan BMS

🚀 Update ke GitHub

1

Edit File

Edit index.html, style.css, atau app.js di folder D:\bms\

2

Push ke GitHub

git add . → git commit -m "update" → git push

3

Live dalam 1-2 Menit

GitHub Pages auto-deploy. Buka neonevadie.github.io/-Baitul-Ma-mur-Safaah/

🔥 Firebase Setup

1

Firestore Rules

Koleksi test = bebas, koleksi lain = wajib login (sesuai screenshot Firebase)

2

Firebase Auth

Aktifkan Email/Password di Authentication → Sign-in method

3

Buat User Owner

Firebase Console → Authentication → Add User → owner@bms-syafaah.id
